# Data Dictionary

This document describes the important columns present in the datasets used in the Mutual Fund Analytics project. It provides information about column names, data types, business meaning, and source datasets.

---

## 1. Fund Master Dataset (`01_fund_master.csv`)

This dataset contains the basic details of all mutual fund schemes.

| Column Name        | Data Type | Business Definition                                                |
| ------------------ | --------- | ------------------------------------------------------------------ |
| amfi_code          | Integer   | Unique AMFI code assigned to each mutual fund scheme.              |
| fund_house         | String    | Name of the mutual fund company managing the scheme.               |
| scheme_name        | String    | Name of the mutual fund scheme.                                    |
| category           | String    | Broad category of the scheme (Equity, Debt, etc.).                 |
| sub_category       | String    | More specific classification within a category.                    |
| plan               | String    | Type of investment plan offered by the scheme.                     |
| launch_date        | Date      | Date on which the scheme was launched.                             |
| benchmark          | String    | Benchmark index used to compare the fund's performance.            |
| expense_ratio_pct  | Float     | Annual fee charged by the fund as a percentage.                    |
| exit_load_pct      | Float     | Charges applicable when redeeming units before a specified period. |
| min_sip_amount     | Integer   | Minimum amount required to start a SIP.                            |
| min_lumpsum_amount | Integer   | Minimum amount required for lump sum investment.                   |
| fund_manager       | String    | Name of the person managing the fund.                              |
| risk_category      | String    | Risk level associated with the scheme.                             |
| sebi_category_code | String    | SEBI classification code of the scheme.                            |

Source: `01_fund_master.csv`

---

## 2. NAV History Dataset (`02_nav_history.csv`)

This dataset stores historical NAV values for mutual fund schemes.

| Column Name | Data Type | Business Definition                                 |
| ----------- | --------- | --------------------------------------------------- |
| amfi_code   | Integer   | Unique identifier of the mutual fund scheme.        |
| date        | Date      | Date on which NAV was recorded.                     |
| nav         | Float     | Net Asset Value of the scheme on a particular date. |

Source: `02_nav_history.csv`

---

## 3. Investor Transactions Dataset (`08_investor_transactions.csv`)

This dataset contains details about investor transactions.

| Column Name        | Data Type | Business Definition                                      |
| ------------------ | --------- | -------------------------------------------------------- |
| investor_id        | String    | Unique identifier assigned to each investor.             |
| transaction_date   | Date      | Date on which the transaction took place.                |
| amfi_code          | Integer   | AMFI code of the scheme involved in the transaction.     |
| transaction_type   | String    | Type of transaction such as SIP, Lumpsum, or Redemption. |
| amount_inr         | Integer   | Amount invested or redeemed in Indian Rupees.            |
| state              | String    | State from which the investor belongs.                   |
| city               | String    | Investor's city.                                         |
| city_tier          | String    | Classification of city based on development tier.        |
| age_group          | String    | Age group of the investor.                               |
| gender             | String    | Gender of the investor.                                  |
| annual_income_lakh | Float     | Annual income of the investor in lakhs.                  |
| payment_mode       | String    | Mode used for making the transaction.                    |
| kyc_status         | String    | Current KYC verification status of the investor.         |

Source: `08_investor_transactions.csv`

---

## 4. Scheme Performance Dataset (`07_scheme_performance.csv`)

This dataset contains performance metrics of mutual fund schemes.

| Column Name       | Data Type | Business Definition                                  |
| ----------------- | --------- | ---------------------------------------------------- |
| amfi_code         | Integer   | Unique identifier of the mutual fund scheme.         |
| return_1yr_pct    | Float     | Return generated by the scheme in the last one year. |
| return_3yr_pct    | Float     | Return generated over the last three years.          |
| return_5yr_pct    | Float     | Return generated over the last five years.           |
| benchmark_3yr_pct | Float     | Three-year return of the benchmark index.            |
| alpha             | Float     | Excess return generated compared to the benchmark.   |
| beta              | Float     | Measure of volatility relative to the market.        |
| sharpe_ratio      | Float     | Risk-adjusted return indicator.                      |
| expense_ratio_pct | Float     | Annual expense charged by the scheme.                |
| risk_grade        | String    | Overall risk rating assigned to the scheme.          |

Source: `07_scheme_performance.csv`

---

## 5. AUM Dataset (`03_aum_by_fund_house.csv`)

This dataset provides information about Assets Under Management (AUM) of different fund houses.

| Column Name    | Data Type | Business Definition                          |
| -------------- | --------- | -------------------------------------------- |
| date           | Date      | Reporting date for AUM values.               |
| fund_house     | String    | Name of the fund house.                      |
| aum_lakh_crore | Float     | Total AUM represented in lakh crore.         |
| aum_crore      | Integer   | Total AUM represented in crore rupees.       |
| num_schemes    | Integer   | Number of schemes managed by the fund house. |

Source: `03_aum_by_fund_house.csv`
